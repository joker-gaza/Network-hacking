# Network-hacking
This is an advanced tool built in Python, developed by Bidaya Company in the Gaza Strip. Designed for beginners in cybersecurity diploma programs and programmed by engineer Ahmed Salim, it is specifically for Linux systems (such as Kali Linux). The tool aims to simplify wireless network penetration testing and communication management via the WinRM and NetworkManager protocols, saving the tester the trouble of typing lengthy commands in the terminal.
